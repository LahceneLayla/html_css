  • * sélectionne TOUS les éléments de ma page html
  • .ma-jolie-nav sélectionne tous les éléments ayant le nom ma-jolie-nav
  • #mon-unique-id selectionne tous les éléments
  • div, article, aside selectionne toutes les balises <div>, <article> et <aside>
  • nav a selectionne les liens <a>inclus/imbriqués dans la balise <nav>
  • nav a:hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriétés CSS utilisé dans la page

Décrivez leur usage et valeurs possibles

  1. background: couleur de fond de l'élément
  2. color: couleur de la police
  3. float: effet flottant
  4. font-family: police
  5. font-size: tailel de la police
  6. height: hauteur de l'élément
  7. left: position a partir de la gauche de la fenêtre
  8. ligne-hight: hauteur de l'interligne
  9. margin: marges exterieurs
  10. padding: marges interieurs
  11. position: position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant)
  12. text-align: alignement du text
  13. text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est suprimé
  14. top: position a partir du haut de la fenêtre
  15. transition: temps de transition lors de l'animation
